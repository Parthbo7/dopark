#### 🚗 Doparking – Smart AI-Based Parking Management System

Doparking is an intelligent, scalable, and real-time parking management platform designed to reduce urban congestion and optimize parking utilization using modern web technologies and smart automation.

---

## 📌 Problem Statement

Urban parking inefficiency leads to:
- Traffic congestion  
- Fuel waste  
- User frustration  

**Doparking solves this by enabling:**
- Real-time parking discovery  
- Smart slot allocation  
- Seamless booking & tracking  

---

## 🚀 Key Features

### 👤 User Panel
- 🔍 Find nearby parking stations  
- 📊 Real-time slot availability  
- 🎟️ Book and reserve slots  
- ⏱️ Live parking timer  
- 💰 Automatic pricing calculation  
- 📥 Download digital ticket  

---

### 🛠️ Admin Panel
- 📡 Monitor all parking slots  
- 📈 View real-time analytics  
- 🔄 Manual/automatic control  
- 💵 Revenue tracking  

---

### 🤖 Smart Automation
- Automatic slot allocation  
- Entry/Exit tracking  
- Dynamic pricing system  
- Future-ready AI integration  

---

## 🧠 Smart Space Optimization

Doparking uses dynamic slot conversion logic:

- 🚗 Car slots ⇄ 🏍️ Multiple bike slots  
- Maximizes parking efficiency in real-time  

---

## 🏢 Frontend Architecture

- **Framework:** React 19  
- **Build Tool:** Vite 8  
- **Routing:** React Router 7  

---

## 🎨 UI & Experience

- **Styling:** Tailwind CSS 3  

**Features:**
- Glassmorphism UI  
- Responsive layouts  
- Smooth transitions  
- Interactive dashboards  
- Clean map-based layout  

---

## ⚙️ Backend & Data

- **Database & Auth:** Supabase (PostgreSQL)  
- **Realtime:** Live slot updates & notifications  

**Logic Layer:**
- PostgreSQL (.sql)  
- Dynamic space optimization  

---

## 🛠️ Utilities

- 🎟️ html-to-image → Ticket generation  
- 📥 downloadjs → File downloads  
- 🔔 Custom Notification System → Real-time alerts  

---

## 📦 Developer Tools

- ESLint 9  
- PostCSS + Autoprefixer  

---

## 🔄 System Workflow

### Automated Mode:
1. Vehicle enters  
2. Slot assigned automatically  
3. Timer starts  
4. Exit detected  
5. Price calculated  
6. Notification sent  

---

| Backend  | Supabase       |
